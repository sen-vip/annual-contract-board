# 📅 연간계약보드

공공요금·연간계약 현황을 한눈에 관리하는 브라우저 기반 업무 보드입니다.

학교 행정실에서 반복적으로 확인해야 하는 공공요금, 연간계약, 자동이체, 만료일, 증빙 상태, 메모 여부를 엑셀 기반으로 정리하고 빠르게 필터링할 수 있도록 만든 보조 도구입니다.

## 🔗 바로가기

- 웹앱: https://sen-vip.github.io/annual-contract-board/
- GitHub: https://github.com/sen-vip/annual-contract-board

## ✨ 주요 기능

- 공공요금, 연간계약, 자동이체 항목 구분 관리
- 엑셀 파일 불러오기
- 정리된 현황 엑셀 내보내기
- 만료 60일 이내 / 만료 30일 이내 항목 빠른 확인
- 종료됨, 메모 있음 항목 필터링
- 세금계산서, 계산서, 지로, 고지서, 무증빙 등 증빙 기준 필터링
- 정렬 기능으로 엑셀 순서, 만료 임박순, 지급일 빠른순 등 확인
- 알림일과 알림내용을 입력해 오늘 확인할 일과 5일 이내 알림 확인
- 캘린더 일정 파일(.ics) 생성
- 브라우저 내 백업 저장 및 백업 불러오기
- 전체 초기화 기능
- 외부 서버 전송 없이 현재 브라우저에만 데이터 저장

## 🧭 사용 흐름

1. `엑셀 불러오기` 버튼으로 계약 현황 자료를 불러옵니다.
2. 빠른보기에서 `전체`, `공공요금`, `연간계약`, `자동이체`, `메모 있음` 항목을 선택해 확인합니다.
3. 필요한 경우 상세필터에서 증빙 종류와 상태를 추가로 선택합니다.
4. 정렬 옵션을 통해 원하는 순서로 목록을 확인합니다.
5. 알림일과 알림내용을 입력하면 앱을 열었을 때 오늘 확인할 일과 5일 이내 알림을 확인할 수 있습니다.
6. 실제 반복 알림이 필요하면 `일정` 버튼으로 캘린더 일정 파일(.ics)을 만들어 Google Calendar 또는 Outlook에 등록합니다.
7. 정리된 자료는 `엑셀 내보내기`로 저장할 수 있습니다.
8. 현재 앱 상태를 보관하려면 `백업 저장`, 이전 상태를 다시 불러오려면 `백업 불러오기`를 사용합니다.

## 🔔 알림 기능 안내

연간계약보드는 서버 없이 브라우저에서 실행되는 정적 웹앱입니다.

따라서 앱이 완전히 닫혀 있거나 컴퓨터가 꺼져 있는 상태에서 매월 자동 푸시 알림을 보장하지 않습니다. 대신 앱을 열었을 때 오늘 또는 곧 확인해야 할 항목을 바로 보여주는 방식으로 설계했습니다.

### 지원하는 알림일 형식

- 매월 1일 ~ 매월 31일
- 매월 말일

예시:

- 매월 5일 / 지출 확인
- 매월 10일 / 자동이체 확인
- 매월 말일 / 납부 확인

`매월 31일`처럼 해당 월에 없는 날짜는 그 달의 말일로 계산합니다.

### 캘린더 일정 파일(.ics)

각 알림 항목의 `일정` 버튼을 누르면 월간 반복 일정 파일을 만들 수 있습니다.
이 파일을 Google Calendar 또는 Outlook에 등록하면 PC나 휴대폰에서 실제 캘린더 알림을 받을 수 있습니다.

## 📂 엑셀 기능과 백업 기능의 차이

### 엑셀 불러오기 / 엑셀 내보내기

계약 현황 자료를 가져오거나, 정리된 현황을 다시 엑셀 파일로 저장할 때 사용합니다.

### 백업 저장 / 백업 불러오기

현재 앱에 저장된 상태를 그대로 보관하거나 복원할 때 사용합니다.
엑셀 자료와는 별도로, 브라우저에 저장된 앱 데이터를 관리하는 기능입니다.

## 🔎 필터 구성

### 빠른보기

자주 확인하는 항목을 빠르게 볼 수 있습니다.

- 전체
- 공공요금
- 연간계약
- 자동이체
- 메모 있음

### 상세필터

필요할 때만 펼쳐서 세부 조건을 확인합니다.

- 증빙: 세금계산서, 계산서, 지로, 고지서, 무증빙
- 상태: 종료됨, 만료 60일 이내, 만료 30일 이내

### 정렬

정렬 기능은 자주 사용하는 기능이므로 빠른보기 오른쪽에 배치했습니다.

- 엑셀 순서
- 만료 임박순
- 구분별 보기
- 계약명 가나다순
- 업체명 가나다순
- 지급일 빠른순
- 계약금액 큰 순
- 월 지급액 큰 순
- 메모 있는 항목 먼저

## 🔒 저장 및 보안 안내

이 웹앱은 별도의 서버로 데이터를 전송하지 않습니다.

불러온 계약 자료와 백업 데이터는 현재 사용 중인 브라우저에만 저장됩니다.
다른 PC나 다른 브라우저에서는 데이터가 자동으로 공유되지 않습니다.

중요한 자료는 필요할 때 엑셀 내보내기 또는 백업 저장 기능으로 별도 보관해 주세요.
