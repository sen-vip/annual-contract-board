# 📅 연간계약보드

공공요금·연간계약 현황을 한눈에 확인하고 관리하는 학교 행정실용 로컬 웹앱입니다.

기존 엑셀로 관리하던 계약 현황을 브라우저에서 불러와서 확인하고, 만료 임박 계약·자동이체·증빙구분·관리메모를 빠르게 볼 수 있도록 만든 보조 도구입니다.

## ✨ 주요 기능

- 공공요금 / 연간계약 / 자동이체 항목 빠른 확인
- 계약명, 업체명, 증빙구분, 메모 통합 검색
- 계약기간 기준 만료상태 자동 표시
- 종료됨 / 만료 60일 이내 / 만료 30일 이내 필터
- 세금계산서, 계산서, 지로, 고지서, 무증빙 필터
- 계약 추가·수정·삭제
- 관리메모 작성 및 메모 있는 항목만 보기
- 엑셀 불러오기 / 엑셀 내보내기
- 백업 저장 / 백업 불러오기
- 브라우저 localStorage 기반 저장

## 🧭 화면 구성

### 상단 영역

- `엑셀 불러오기`: 기존 계약현황 엑셀 파일을 가져옵니다.
- `계약 추가`: 엑셀 없이 계약 항목을 직접 추가합니다.
- `엑셀 내보내기`: 현재 계약 목록을 업무용 엑셀 파일로 저장합니다.
- `백업 저장`: 현재 앱 상태를 백업 파일로 저장합니다.
- `백업 불러오기`: 저장해둔 백업 파일로 앱 상태를 다시 불러옵니다.
- `전체 초기화`: 현재 브라우저에 저장된 계약 데이터를 삭제합니다.
- `GitHub`: 저장소로 이동합니다.

### 빠른보기와 정렬

자주 쓰는 필터와 정렬을 상단에 배치해 화면을 덜 복잡하게 정리했습니다.

- 전체
- 공공요금
- 연간계약
- 자동이체
- 메모 있음
- 정렬: 엑셀 순서, 만료 임박순, 구분별 보기, 계약명 가나다순 등

### 상세필터

필요할 때만 펼쳐서 사용하는 필터입니다.

- 증빙: 세금계산서, 계산서, 지로, 고지서, 무증빙
- 상태: 종료됨, 만료 60일 이내, 만료 30일 이내

## 🔒 보안 안내

이 앱은 서버 전송 없이 브라우저 안에서 동작합니다.

- 입력한 데이터는 현재 브라우저에 저장됩니다.
- 엑셀 파일은 브라우저에서만 읽습니다.
- 외부 서버로 계약 데이터가 전송되지 않습니다.
- 다른 PC나 다른 브라우저에서는 데이터가 자동으로 공유되지 않습니다.

## 💾 엑셀과 백업의 차이

- `엑셀 불러오기 / 엑셀 내보내기`: 사람이 열어보는 업무용 계약현황 파일입니다.
- `백업 저장 / 백업 불러오기`: 앱 상태를 그대로 보관하고 다시 복원하기 위한 파일입니다.

업무 제출이나 공유가 목적이면 엑셀을 사용하고, 내 PC에서 쓰던 앱 상태를 보관하려면 백업을 사용하면 됩니다.

## 🛠 사용 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. 기본 계약현황을 확인하거나 `엑셀 불러오기`로 기존 현황표를 가져옵니다.
3. 계약 항목을 클릭해 내용을 확인하거나 수정합니다.
4. 자주 보는 항목은 `빠른보기`로 확인합니다.
5. 정렬은 빠른보기 오른쪽에서 바로 변경합니다.
6. 증빙, 종료 여부, 만료 임박 항목은 `상세필터`를 펼쳐 확인합니다.
7. 필요하면 `엑셀 내보내기` 또는 `백업 저장`으로 보관합니다.
