# 🚌 버스콕검 v0.2.38

전세버스 임차 계약 서류를 담당자가 직접 확인할 수 있도록 정리해주는 **체크리스트형 검토 보조 웹앱**입니다.

버스콕검은 자동으로 최종 적합/부적합을 판정하는 도구가 아닙니다.
업로드한 제출파일을 참고용으로 분류하고, 담당자가 실제 서류에서 확인해야 할 항목을 **계약조건별 → 업체확인서류 → 차량검토서류 → 차량별필수서류** 흐름으로 정리해주는 도구입니다.

배포 페이지:
https://github.com/sen-vip/bus-kockgum

---

## ✨ 주요 기능

* 출발일 기준 보험기간·검사유효기간 확인 보조
* 추정가격 기준 확인 필요 서류 안내
* 배정차량 리스트 자동 추출 및 수동 보정
* 업로드한 제출파일의 서류 후보 자동 정리
* 제출파일별 관련 페이지 후보 표시
* 계약조건별 서류, 업체확인서류, 차량검토서류, 차량별필수서류 흐름 정리
* 차량별 자동차등록원부, 자동차 종합보험 가입증명서, 전세버스 교통안전정보 조회결과 통보서, 출발 전 교육 및 차량안전점검표 확인
* 차량번호와 운전자 정보 확인
* 담당자가 직접 체크할 수 있는 체크박스 제공
* 부족 서류에 대한 보완요청 문구 정리
* 결과 내보내기 CSV 제공
* 브라우저 기반 처리로 업로드 파일 외부 전송 없음

## 🧭 검토 흐름

### 1. 출발정보 입력

출발일을 먼저 입력합니다.
출발일은 보험기간, 검사유효기간, 출발 전 점검표 확인의 기준이 됩니다.

추정가격은 선택 입력 항목이지만, 계약 조건에 따라 추가 확인이 필요한 서류 판단에 참고됩니다.

### 2. 차량 정보 확인

배정차량 목록을 입력하거나 제출파일에서 추출된 차량번호를 확인합니다.

차량번호와 운전자 정보를 기준으로 차량별 필수서류 확인 흐름이 만들어집니다.

### 3. 제출파일 업로드

업체가 제출한 PDF 또는 이미지 파일을 업로드합니다.

업로드한 파일은 참고용으로 분류되며, 어떤 파일이 어떤 서류 후보에 가까운지 `제출파일 정리` 영역에서 확인할 수 있습니다.

### 4. 필요서류 체크리스트 확인

필요서류는 아래 흐름으로 정리됩니다.

1. 계약조건별
2. 업체확인서류
3. 차량검토서류
4. 차량별필수서류

각 항목에서는 다음 내용을 확인할 수 있습니다.

* 이 서류가 어디에 있는지
* 이 서류에서 뭘 봐야 하는지
* 담당자가 해야 할 일
* 업체에 보완요청할 문구

### 5. 차량별 체크

각 차량별로 제출 여부와 확인 항목을 점검합니다.

* 자동차등록원부
* 자동차 종합보험 가입증명서
* 교통안전정보
* 출발 전 교육 및 차량안전점검표

### 6. 결과 내보내기

검토 결과는 CSV로 내보낼 수 있습니다.

CSV는 계약조건별, 업체확인서류, 차량검토서류, 차량별필수서류 흐름에 맞춰 정리됩니다.

## 📁 확인 서류 범위

### 계약조건별

* 직접생산확인증명서
* 중소기업확인서
* 기타 계약 조건에 따라 필요한 서류

### 업체확인서류

* 여객자동차 운송사업등록증
* 사업자등록증
* 직영차량 운행각서

### 차량검토서류

* 차량 및 운전기사 배정 현황표
* 차량보유 현황표 (선택 구비서류)

### 차량별필수서류

* 자동차등록원부
* 자동차 종합보험 가입증명서
* 전세버스 교통안전정보 조회결과 통보서
* 출발 전 교육 및 차량안전점검표

## 🏷️ 상태값 안내

| 상태        | 의미                           |
| --------- | ---------------------------- |
| 제출 확인됨    | 제출된 서류에서 해당 내용을 확인할 수 있음     |
| 관련 페이지 있음 | 관련 내용이 있을 가능성이 높은 페이지 후보가 있음 |
| 담당자 확인필요  | 자동 판단이 어려워 담당자가 직접 확인해야 함    |
| 못 찾음      | 제출파일에서 관련 내용을 찾지 못함          |
| 보완 필요     | 제출 또는 확인 보완이 필요함             |

## 🔎 화면 구성

### 입력 영역

출발일, 추정가격, 차량정보, 제출파일을 입력하는 영역입니다.

출발일은 필수 확인 기준이며, 추정가격은 계약조건별 확인에 참고됩니다.

### 제출파일 정리

업로드한 파일이 어떤 서류 후보로 분류되었는지 참고용으로 보여줍니다.

파일명, 서류 후보, 관련 페이지 후보를 함께 확인할 수 있습니다.

### 검토요약

전체 제출 상태와 보완 필요 항목을 한눈에 확인할 수 있습니다.

### 필요서류 체크리스트

계약 담당자가 실제로 확인해야 하는 서류를 업무 흐름에 맞게 정리합니다.

### 차량별 체크

차량번호별로 필수서류 제출 여부와 확인해야 할 내용을 정리합니다.

## 🔐 보안 안내

버스콕검은 업로드한 PDF·이미지 파일을 현재 브라우저에서 처리합니다.

파일 내용은 외부 서버로 전송하지 않습니다.

PDF 텍스트 추출이나 이미지 글자 판독을 위해 PDF.js, Tesseract.js 같은 라이브러리를 CDN에서 불러올 수 있습니다.
이는 분석 도구를 불러오는 과정이며, 사용자가 업로드한 파일 내용이 외부로 전송되는 것은 아닙니다.

## ⚠️ 주의사항

* 자동 결과는 행정 검토 보조용입니다.
* 최종 확인은 담당자가 직접 해야 합니다.
* 스캔본, 흐린 이미지, 기울어진 문서, 도장이나 스탬프가 겹친 파일은 정확도가 떨어질 수 있습니다.
* 보험기간, 검사유효기간, 실제 배차 차량, 실제 운전자, 출발 전 점검 여부는 원본 확인이 필요할 수 있습니다.
* 교통안전정보 조회결과 통보서의 진위 여부는 별도 확인이 필요합니다.
* `관련 페이지 있음`은 확정 판정이 아니라 확인 후보 안내입니다.
* `담당자 확인필요` 항목은 원본 서류를 직접 확인해야 합니다.
