버스콕검

전세버스 임차 서류 체크리스트

제출한 서류에서 확인이 필요한 항목을 체크리스트로 정리해드려요. 담당자가 놓치기 쉬운 부분을 순서대로 확인할 수 있도록 돕는 검토 보조 도구입니다.

📋 배정차량 기준 확인 📄 PDF 쪽수 안내

STEP 1

출발일 입력

필수

출발일은 꼭 필요해요. 보험기간과 검사유효기간을 이 날짜 기준으로 확인합니다.

필수 입력

서류 유효기간 확인 기준일입니다.

안내 대기

추정가격을 입력하면 직접생산확인증명서 확인 필요 여부를 안내합니다.

추정가격 입력 시 1천만원 기준 안내가 표시됩니다.
선택 정보 입력하면 결과 안내가 더 자연스러워져요 ▾

STEP 2

배정차량 입력

차량번호 추출용

차량 및 운전기사 배정 현황표 등 차량번호가 있는 기준자료를 먼저 올려주세요. 파일이 없으면 직접 입력해도 됩니다.

📋

배정차량 파일을 여기에 올려주세요

PDF · 이미지 · 드래그하거나 클릭

아직 업로드된 파일이 없어요

STEP 3

배정차량 리스트 확인

대기 중

추출된 차량번호를 확인하고 틀린 내용은 직접 수정하세요. 이 목록이 콕검의 기준이 됩니다.

STEP 2에서 파일을 업로드하고 추출하기 버튼을 눌러주세요.
스캔본이라 자동으로 찾기 어려운 경우 아래 버튼으로 직접 입력할 수 있어요.
호차 차량번호 운전자 연락처 비고 삭제

STEP 4

서류 업로드

보험·등록·교통안전 등

배정차량 파일을 제외한 나머지 서류를 올려주세요. 스캔본도 괜찮아요 — 다만 글자를 자동으로 읽기 어려운 이미지 중심 서류는 없는 서류로 처리하지 않고, 확인해야 할 PDF 쪽수 후보를 안내합니다.

계약조건별 서류
  • 중소기업확인서
  • 직접생산확인증명서
업체 확인 서류
  • 여객자동차 운송사업등록증
  • 사업자등록증
  • 직영차량 운행각서
차량검토 기준자료
  • 차량 및 운전기사 배정 현황표
  • 차량보유 현황표 (선택 구비서류)
차량별 필수서류
  • 자동차등록원부
  • 자동차 종합보험 가입증명서
  • 전세버스 교통안전정보 조회결과 통보서
  • 출발 전 교육 및 차량안전점검표
🖇️

PDF 서류를 여기에 끌어다 놓거나 클릭하세요

여러 파일을 한 번에 올릴 수 있어요

텍스트 우선 · 필요 시 자동판독

아직 업로드된 파일이 없어요

STEP 5

콕검 실행

대기 중

텍스트형 PDF를 먼저 분석하고, 필요한 경우 체크리스트에서 자동판독 보완을 선택할 수 있어요.

🔍

아직 콕검을 실행하지 않았어요

출발일 입력 → 배정차량 입력 → 차량번호 추출 확인 → 나머지 서류 업로드 → 콕검 시작

필요하면 담당자 상황에 맞게 문구를 직접 수정한 뒤 복사해서 사용하세요.

버스콕검은 제출서류의 주요 항목 확인을 돕는 보조 도구입니다. 교통안전정보 통보서 진위확인, 실제 배차 차량 확인, 실제 운전자 확인, 출발 전 차량 안전점검은 담당자가 직접 확인해야 합니다. 이 앱은 법적 적합성을 최종 판정하지 않습니다.
